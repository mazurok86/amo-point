track.js — локальная проверка

Эта страница подключает track.js через <script async src="/track.js">. При загрузке отправляется один POST на /api/visits.

Как проверить

  1. Откройте DevTools (F12) → вкладка Network, фильтр visits.
  2. Перезагрузите страницу.
  3. Должен пройти POST /api/visits со статусом 204 No Content.
  4. Body запроса (Form Data): visitor_uid, page_url, user_agent (+ referrer если есть).
  5. В консоли проверьте localStorage.getItem('__amo_visit_uid') — должен быть UUID v4. Перезагрузите страницу — UUID должен сохраниться.
  6. В таблице visits должна появиться запись (через php artisan tinker → Visit::latest()->first()).
